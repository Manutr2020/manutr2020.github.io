Musical Search Engine

A web-based information retrieval system for a musical corpus, combining Boolean search, TF-IDF ranking and semantic search.

Overview

The Musical Search Engine allows users to search within a corpus of musical descriptions using multiple retrieval strategies.

Features

  • Boolean search with AND, OR and NOT
  • Wildcard search support
  • TF-IDF ranking
  • Semantic similarity search
  • Theme extraction from musical descriptions
  • Web interface for query input and result display

My Contribution

I contributed to the Boolean Search engine, Semantic Search engine, wildcard search functionality, theme extraction module, and parts of the Flask application refactoring.

Tools

Python · Flask · NLTK · scikit-learn · HTML · CSS · semantic similarity
